◆ ParseTimespan()

def seims.run_seims.MainSEIMS.ParseTimespan (   self) 
The format of self.timespan is different for OpenMP version and MPI&OpenMP version.
For OpenMP version:
   {'IO': {'Input': 0.2,
           'Output': 0.04
          }
    'COMP': {'TSD_RD_P': 0.0001,  # All modules
             'ALL': 12.3
            }
    'SIMU': {'ALL': 14.1}
   }
For MPI&OpenMP version:
   {'MAX': {'IO': {'Input': 0.1,
                   'Output': 0.02,
                   'ALL': 0.12
                  }
            'COMP': {'Slope': 5,
                     'Channel': 0.5,
                     'Barrier': 0.1,
                     'ALL': 5.6
                    }
            'SIMU': {'ALL': 10.1}
           }
    'MIN': {...}
    'AVG': {...}
   }
